What I Do

I can add value and help alleviate pain points in the following areas:

Web Development

Build websites from the ground up. I've built websites in HTML, WordPress, Shopify, Sqaurespace, React, Angular, Vue etc. All sites are built with clean reuseable and scaleable code. All sites will work in all modern web browsers.

Web Applications
